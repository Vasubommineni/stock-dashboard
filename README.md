# 📊 Stock Dashboard

## 🚀 Overview
This project is a stock data dashboard that fetches real-time stock prices and visualizes them.

## ⚙️ Tech Stack
- Python (Flask)
- Pandas
- yFinance
- Chart.js

## 📌 Features
- Fetch stock data (INFY, TCS)
- Display closing prices
- 7-day Moving Average
- Interactive chart visualization

## ▶️ How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Run:
python app.py

3. Open:
https://stock-dashboard-64fz.onrender.com
